== README

This is a recipe builder for brewing beer. It allows the user to compare metrics for given recipes and take notes for future brew days. It also allows brewers to share and comment on each others recipes. Ideally, it is a process productivity tool, allowing the brewer to gain by his experience and the experience of others.

* Ruby version
2.1.5

* Database creation
postgres database

* Database initialization
Run rake db:create
		rake db:migrate
		rake db:seed
